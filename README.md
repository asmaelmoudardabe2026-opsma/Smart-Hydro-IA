# Smart-Hydro IA & Backend Engine 🌿

Ce dépôt contient le moteur d'intelligence logicielle et l'API Backend du projet **Smart-Hydro**.

## 🚀 Fonctionnalités
- **Algorithme IA :** Calcul précis des besoins en eau basé sur la formule scientifique $Besoin = ETo \times Kc$.
- **Backend API :** Développé avec **FastAPI** pour une réponse rapide et asynchrone.
- **Data :** Intégration simulée des flux météo (OpenWeatherMap) et gestion des coefficients de culture marocains.
